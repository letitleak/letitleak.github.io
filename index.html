Biography

Hi! I’m Yoha誰 Berreby, or @yberreby for short. I am a self-taught developer.

When I’m not studying, I spend time studying programming, hacking around, reading technical articles, mainly on Hacker News and /r/programming. I started being interested in programming around 2011.

I first dabbled with C, then went on to learn some Python, Ruby, client- and server-side (with Node.js) JavaScript. But these days, I’m mainly using Rust, a promising systems programming language backed by Mozilla featuring zero-cost abstractions, a very strong type system and compile-time prevention of memory unsafety and data races.

In the past, I wrote a few Node.js bots to interface with Steam trading, using the node-steam NPM module.
